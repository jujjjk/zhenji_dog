ROS2 真机控制可先定义：

```bash
cmd() {
  ros2 topic pub -r 10 /cmd_vel geometry_msgs/msg/Twist \
    "{linear: {x: $1, y: $2, z: 0.0}, angular: {x: 0.0, y: 0.0, z: $3}}"
}
```

然后执行：

```bash
cmd 0.15 0 0          # 前进
cmd -0.05 0 0         # 后退
cmd 0 0.04 0          # 左横移
cmd 0 -0.04 0         # 右横移
cmd 0 0 0.30          # 原地左转
cmd 0 0 -0.30         # 原地右转
cmd 0.12 0 0.25       # 前进左转
cmd 0.12 0 -0.25      # 前进右转
cmd -0.04 0 0.15      # 后退左转
cmd 0.08 0.04 0       # 左前斜移
cmd 0.08 -0.04 0      # 右前斜移
```

按 `Ctrl+C` 停止持续发布，随后发送：

```bash
cmd 0 0 0
```
